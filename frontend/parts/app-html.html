How AI provider selection works
The server picks the provider in this order when an AI task runs:
1. Your preferred provider (set above) — if an API key is configured and budget remains
2. Anthropic Claude — if key present and under budget
3. OpenAI GPT — if key present and under budget
4. Workers AI — always available, free, lower accuracy
What each AI task does:
Price Analysis — generates STR/LTR pricing strategies from comps + market data
Generate Strategy — full PriceLabs-ready strategy: base/min/max, discounts, min nights, setup steps
Optimize Revenue — action plan: quick wins, occupancy gaps, ADR improvements, 90-day plan
Acquisition Analysis — ROI, cap rate, cash-on-cash, buy/hold/pass recommendation
Market Deep Dive — narrative market analysis from snapshot data
